# 🤝 B2B Partnership Showcase

A user-friendly and scalable Flutter-based B2B marketplace application designed to connect businesses, service providers, and clients through a structured digital platform. The app enables users to explore services, browse categories, submit requests, manage applications, and track business opportunities through a clean and intuitive workflow.

---

## 👤 User Features

- Browse business categories and service providers  
- Explore detailed business profiles and opportunities  
- Submit service requests or apply for opportunities  
- Track request and application status  
- Clean and responsive UI/UX experience  
- Multi-language support (Arabic & English)

---

## 🛠️ Admin Features

- Manage business categories and listings  
- Review incoming requests and applications  
- Approve, reject, or organize submissions  
- Manage platform workflow efficiently  

---

## 🔄 App Flow

1. User browses categories and services  
2. User opens provider/business details  
3. User submits request or application  
4. Admin reviews submission  
5. User receives status updates in real-time  

---

## ✨ Key Features

- Authentication (Login - Register - Reset Password - OTP Verification - Profile Management)  
- Business listings & categories browsing  
- Provider / company detailed profiles  
- Request & application management system  
- Favorites & saved opportunities  
- Advanced search & filtering system  
- Real-time status tracking  
- Push notifications  
- Multi-language support (Arabic & English)  
- Admin management dashboard
